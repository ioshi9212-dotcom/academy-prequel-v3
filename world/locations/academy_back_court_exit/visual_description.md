# Задний выход у баскетбольных площадок

Серое небо после ночного дождя. Асфальт у площадок влажный, местами темнее под сеткой. Прохладный ветер цепляет волосы и края одежды. Через задний вход поток студентов меньше, но площадки рядом делают место заметным: мяч, голоса, металлический звон кольца, взгляды у сетки.

Локация подходит для короткого социального давления: студенты смотрят, Хару связан с площадкой, Райден может быть заметен как холодное молчаливое присутствие на фоне.
